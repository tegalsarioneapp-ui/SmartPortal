BA lengkap Pdf
*"Tolong buatkan template HTML & CSS statis untuk Laporan Keuangan (Riwayat Transaksi). Pastikan struktur HTML semantik, tanpa menggunakan Flexbox/Grid (gunakan display: table), dan styling menggunakan CSS internal agar mudah di-render menjadi PDF (misal dengan Puppeteer / wkhtmltopdf) TANPA ada teks atau baris tabel yang terpotong di antara dua halaman (hindari page break issue).
PAGE SETUP & TYPOGRAPHY:
Ukuran: A4 (210mm x 297mm), orientasi Portrait, box-sizing: border-box.
Margin halaman: 2cm (@page { margin: 2cm; size: A4 portrait; }).
Font utama: Arial, sans-serif. Warna teks: #000000. Line-height: 1.15.
Ukuran font body: 12px.
Ukuran judul dokumen: 14px (Bold, Uppercase, rata tengah).
KOP SURAT (HEADER):
Layout menggunakan display: table. Kolom kiri (15%) untuk Logo, Kolom kanan (85%) rata tengah untuk Teks.
Baris 1 & 2 (Pemerintah & Kecamatan): font 16px bold.
Baris 3 (Nama RT/RW): font 14px bold.
Baris 4 (Alamat): font 12px normal.
Garis bawah ganda (border-bottom: 3px double #000; padding-bottom: 10px; margin-bottom: 20px).
TABEL DATA:
Lebar 100%, border-collapse: collapse, border sel 1px solid #000, padding sel 8px.
Background table header (th): #f2f2f2, teks bold, rata tengah.
[PENTING - ANTI TERPOTONG]: Gunakan thead { display: table-header-group; } agar header berulang di halaman 2. Gunakan tr { page-break-inside: avoid; break-inside: avoid; } pada CSS agar baris data tidak terpotong setengah jika kena pindah halaman.
AREA TANDA TANGAN:
Layout menggunakan display: table dengan lebar 100%. Dua kolom sejajar (display: table-cell; width: 50%; text-align: center; vertical-align: top;), tanpa border.
Berikan ruang vertikal tinggi 80px di bawah jabatan untuk ruang tanda tangan basah.
Nama terang dibold dan digarisbawahi.
[PENTING - ANTI TERPOTONG]: Bungkus area tanda tangan ini dan berikan CSS page-break-inside: avoid; agar satu blok tanda tangan tidak terpisah di halaman yang berbeda.
FOOTER:
Teks ukuran 10px, italic, #666666, letakkan rata tengah (fixed position) di bagian paling bawah halaman."
